## Installation

In hubot project repo, run:

`npm install hubot-lunchroulette --save`

Then add **hubot-lunchroulette** to your `external-scripts.json`:

```json
[
  "hubot-lunchroulette"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
